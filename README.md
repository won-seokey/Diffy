# 🔍 Diffy

**Diffy**는 두 텍스트, 코드, 파일 간의 차이점을 쉽고 빠르게 찾아주는 직관적인 라이트웨이트 diff checker 웹앱입니다. 복잡한 과정 없이 비교하고 싶은 내용을 입력하면, 추가(Green)되거나 삭제(Red)된 부분을 한눈에 명확하게 시각화하여 보여줍니다.

### ✨ 주요 기능
- **실시간 비교:** 두 텍스트나 코드를 Side-by-Side(좌우) 또는 Inline(통합) 방식으로 즉시 비교합니다.
- **명확한 시각화:** 수정, 추가, 삭제된 라인을 직관적인 컬러 하이라이팅으로 제공합니다.
- **사용자 친화적 UI:** 일반 사용자부터 개발자까지 누구나 쉽게 쓸 수 있는 미니멀하고 깔끔한 화면을 제공합니다.
- **빠른 성능:** 대용량 텍스트나 파일도 지연 없이 매끄럽고 빠르게 분석합니다.
